READ-LEVEL RAREFACTION SENSITIVITY CHECK — alpha diversity (CORAL_P clean-10)
============================================================================
Design : host-removed reads (R1+R2+U) of each of the 30 clean-10 libraries were
         subsampled to a COMMON 13,000,000 reads/library (seqkit '-p' Bernoulli
         subsampling, fixed seed 42, then truncated to 13M -> equal depth; a
         fixed-depth subsample, not a uniform exact-size SRS), then re-profiled
         with MetaPhlAn 4 (same DB/params as s4). Alpha metrics recomputed on the
         rarefied profiles and compared to native depth.
Basis  : rarefy INPUT reads, not the profile table (N. Segata, MetaPhlAn dev,
         metaphlan-users forum); rarefaction matters mainly for richness
         (Liu et al. 2022, Genome, PMID 35939836); MetaPhlAn -> relative abundance
         (Blanco-Miguez et al. 2023, Nat Biotechnol). Generated by
         scripts/rarefaction_sensitivity.sbatch + scripts/rarefaction_alpha.R.
============================================================================
compared 30 clean-10 samples (native vs rarefied to 13M reads)

===== Observed =====
  core    native median =  163.500    rarefied median =  129.000
  shell   native median =  185.500    rarefied median =  165.500
  stool   native median =  130.500    rarefied median =  116.500
  compartment ranking : native [shell > core > stool]  vs  rarefied [shell > core > stool]  -> PRESERVED

========== PAIRWISE POST-HOC (paired Wilcoxon, per participant, n=10; uncorrected) ==========
  Observed    native  cs=0.083 cS=0.049 Ss=0.009 | rarefied  cs=0.169 cS=0.019 Ss=0.010  (cs=core-stool, cS=core-shell, Ss=shell-stool)
  Shannon     native  cs=0.557 cS=0.006 Ss=0.014 | rarefied  cs=0.557 cS=0.006 Ss=0.014  (cs=core-stool, cS=core-shell, Ss=shell-stool)
  InvSimpson  native  cs=1.000 cS=0.010 Ss=0.027 | rarefied  cs=0.922 cS=0.010 Ss=0.027  (cs=core-stool, cS=core-shell, Ss=shell-stool)
  -> core-stool (cs) is n.s. for every metric; the omnibus signal is the SHELL.

Interpretation: Shannon/InvSimpson ~unchanged (depth-robust); Observed drops uniformly under
rarefaction but its compartment ranking + Friedman significance should persist if the native
richness comparison is not explained by sequencing depth alone.

CONCLUSION
----------
The alpha-diversity results are NOT explained by sequencing depth. Shannon and
inverse Simpson are essentially unchanged under equal-depth subsampling
(per-sample Spearman rho = 0.999; identical Friedman p = 0.025), confirming the
relative-abundance primary analysis is depth-robust. Observed richness drops
uniformly with fewer reads (as expected) but retains its compartment ranking
(shell > core > stool), per-sample rank (rho = 0.956), and significance
(Friedman p = 0.045 in both). Pairwise post-hoc: the signal is the SHELL
(core-shell and shell-stool significant); core vs stool is NOT significant on any
metric. So the shell's higher diversity persists at equal depth -- this supports,
but does not by itself prove, a biological (rather than technical) origin.
